What is WeebReact?
WeebReact is an Application based on the
Electron Framework
that is made to give you optimal control over your Reaction Pics.
Features
-
Tagging: Assign tags to your pictures to easily find them
-
Search: Search for pictures by including the tags you want to see and excluding those you don't
(Exclude tags by right-clicking on them in the search autocomplete suggestions or by just clicking on them in the search tag list)
-
Sorting: Sort pictures by the time you added them, by filename or by rating, in both ascending and descending order
-
Rating: Rate pictures from 1 to 10 depending on how much you like them! Always see your favorite pictures first
-
NSFW Exclude: Got some lewd stuff? You can choose to exclude pictures that have the "nsfw" tag and only show them when you search for "nsfw" so no one accidentally sees your hidden gems
-
Drag'n'Drop: Just Drag and Drop any picture you want to use out of the program to wherever you need it
(Doesn't work with discord, however, that's a problem on discord's side. Fix your shit discord.)
-
Reverse Drag'n'Drop: If there's a picture in your folder that you can't find in WeebReact because it's still untagged, just drag it into WeebReact to open it's detail page and tag it
-
Multi-Selection: Select multiple pictures at once and add or remove tags from all of them. Cuz let's be honest we're all too fucking lazy to tag each pic one by one.
-
Multi-Directory support: Got your pictures spread across multiple folders? No problem, include as many folders as you want!
-
Directory watcher: No need to manually restart or reload the program if you added new pics, WeebReact pics up any changes automatically.
(You can disable this on a per-directory basis, if a directory has a lot of subdirectories this can cause heavy cpu usage)
-
Filter: Feel like posting a .gif this time? No problem, you can filter for filetypes! Supported filetypes are: jpg png gif webm mp4
-
List all untagged pictures: Got a lot of pics and feel like you missed some of them
-
Zoom: Want to see your pictures a bit.. closer? Adjust the size you see them in! (50% to 200%)
